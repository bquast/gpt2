relay & query

how this works (short)

this page opens a WebSocket to the relay and sends a REQ with filters: { kinds: [KIND], limit: N, "#tag": ["value"] } if a tag is provided. it lists the newest events; each article uses the first non-empty line as a title, and the rest as body.
